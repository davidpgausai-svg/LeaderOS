# ‚≠ê **Your App.yaml Must Support One-Click**

Make sure your `.do/app.yaml` contains:

* persistent SQLite volume
* port 3000
* build + run commands
* env var definition

Example (simplified):

```yaml
name: strategicflow
services:
  - name: app
    github:
      repo: YOURNAME/strategicflow
      branch: main
    http_port: 3000
    envs:
      - key: JWT_SECRET
        scope: RUN_AND_BUILD_TIME
    build_command: "npm install && npm run build"
    run_command: "npm start"
    volumes:
      - name: sqlite-data
        mount_path: /data

volumes:
  - name: sqlite-data
```

This gives true 1-click deployability.
